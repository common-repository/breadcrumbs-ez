Lightweight Breadcrumb Plugin.
Add a breadcrumbs shortcode  any where in your theme.

== Description ==

Add the code to your theme: echo do_shortcode('[breadcrumbs]');<br /><br />
Alternatively, you can also use the [breadcrumbs] shortcode if implementing in a single post or page.<br /><br />
Contorl the wraping HTML tags and the separator.


== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
